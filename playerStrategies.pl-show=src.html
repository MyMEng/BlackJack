dealType(-Type, +Player) is det
Check the hand type of given player:

    

  • hard - if it does not contain any ace;
    • 

  • soft - if it contains at least one ace.
    • 






Arguments:





Type- The type of the player's hand: hard or soft.


Player- Player's cards: a list of card predicates.












  27dealType(Type, Player) :-
  28	% count the number of aces
  29	aggregate_all(count, member(card(_,ace), Player), Aces),
  30	( Aces = 0  -> Type = hard 
  31	; otherwise -> Type = soft
  32	).

playAISequence(-NewTable, -NNDeck, -NRefuse, +Table, +NDeck, +Dealer, +Counter, +X, +Refuse) is det
Assigns strategy to each player and simulates then; then returns resulting:
table, deck, refusal list. The first player in the list plays playAIDet,
the second playAIDeck, the third playAIMix, the fourth always holds,
finally the rest of the players are playing playAIDet strategy.




Arguments:





NewTable- New list of cards held by all players excluding dealer
after current round.


NNDeck- New list of cards remaining in the deck after current round.


NRefuse- New list of players not allowed to draw a card after
current round.


Table- List of cards held by all players excluding dealer* before
current round.


NDeck- List of cards remaining in the deck before current round.


Dealer- Dealer's cards: a list of card predicates.


Counter- The loop counter.


X- The total number of AI driven players.


Refuse- List of players not allowed to draw a card before current
round.












 150playAISequence(Table, Deck, Refuse, Table, Deck, _, X1, X, Refuse) :-
 151	X1 is X +1,
 152	!.
 153playAISequence(NewTable, NNDeck, NRefuse, Table, NDeck, Dealer, Counter, X, Refuse) :-
 154	% player B is,
 155	nth1(Counter, Table, PlayerB),
 156
 157	% do different strategy for each player
 158	nth1(Counter, Refuse, Status),
 159	( Status = 0 ->
 160		( Counter = 1 -> ( playAIDet( PlayerA, ChDeck, PlayerB, NDeck, Dealer ),
 161						   replace( NewRefuse, Refuse, 1, Counter) )
 162		; Counter = 2 -> ( playAIDeck( PlayerA, ChDeck, PlayerB, NDeck, Table), NewRefuse = Refuse )
 163		; Counter = 3 -> ( playMix( PlayerA, ChDeck, PlayerB, NDeck, Dealer, Table ),
 164		                   replace( NewRefuse, Refuse, 1, Counter) )
 165		; Counter = 4 -> ( PlayerA = PlayerB, ChDeck = NDeck, replace( NewRefuse, Refuse, 1, Counter) )
 166		; otherwise   -> ( playAIDet( PlayerA, ChDeck, PlayerB, NDeck, Dealer ),
 167						   replace( NewRefuse, Refuse, 1, Counter) )
 168		)
 169	; otherwise ->
 170		( PlayerA = PlayerB, ChDeck = NDeck, NewRefuse = Refuse )
 171	),
 172
 173	% if AI once refused do not allow any more
 174	( ( PlayerA = PlayerB, Counter = 2) -> replace( NewNewRefuse, NewRefuse, 1, Counter) % refusal
 175	; otherwise                         -> NewNewRefuse = NewRefuse % action taken
 176	),
 177
 178	% attach back player A.
 179	replace( ChTable, Table, PlayerA, Counter),
 180
 181	Counter1 is Counter + 1,
 182	playAISequence(NewTable, NNDeck, NRefuse, ChTable, ChDeck, Dealer, Counter1, X, NewNewRefuse).

playAIDeck(-PlayerA, -ChDeck, +PlayerB, +Deck, +Table) is det
2nd strategy playAIDeck: Shuffle tracking strategy which estimates deck
probabilities. It assumes that the deck has started being in ordered; then
it calculates probabilities of each card occurring at given position and
decides to hit or stand based on the cards with highest probabilities.




Arguments:





PlayerA- Player's hand after player's action.


ChDeck- List of cards in the deck after player's action.


PlayerB- Player's hand before player's action.


Deck- List of cards in the deck before player's action.


Table- List of cards held by all players excluding dealer* before
current round (list of lists).

initDeckProbabilities(-Deck, +Quantity, +Type) is det
Simulate card shuffle by preparing deck probability table for initial number
of shuffles. The type of shuffle can be one of those defined by shuffleMode
parameter in deck package. The binary vector (list of 0's and 1's) that
decides probability of given card position in shuffle is estimated based on
shuffle type.




Arguments:





Deck- A shuffled initial deck of cards (list of predicates card).


Quantity- Total umber of shuffles to do.


Type- Type of card shuffle to simulate.












 261initDeckProbabilities(Deck, Quantity, random) :-

 268	),

shuffleMock(-Shuffled, +Deck, +N, +Sampling) is det
Simulate card shuffle; the type of shuffle to simulate is taken from system
settings: see shuffleMode parameter in deck package.




Arguments:





Shuffled- Shuffled deck of cards (list of predicates card).


Deck- A deck of cards to be shuffled (list of predicates card).


N- Shuffling counter - number of shuffles to do.


Sampling- A binary vector (list of 0's and 1's) that decides whether
card from given pile (pile A) should be placed on top or
bottom of the card from other pile (pile B).












 291shuffleMock(Shf, Shf, 0, []) :- !.
 292shuffleMock(Shuffled, Deck, N, Sampling) :-
 293	shuffleMode(Mode),
 294	proper_length(Deck, Len),
 295	Half is Len / 2,
 296	getPiles(A, _, Half),
 297	A1 is A + 1,
 298	split(P1, P2, Deck, A1), % get two piles
 299	% random or deterministic
 300	( Mode = random        -> rifleRanMock(Forward, P1, P2, NSampling, Sampling)
 301	; Mode = deterministic -> (rifleDetMock(Forward, P1, P2), NSampling = Sampling )
 302	),
 303	N1 is N - 1,
 304	shuffleMock( Shuffled, Forward, N1, NSampling ).

3rd strategy playMix: 0.5 chance of playing deterministic strategy (1st) and
0.5 chance of playing deck memory strategy (2nd).




Arguments:





PlayerA- Player's hand after player's action.


ChDeck- List of cards in the deck after player's action.


PlayerB- Player's hand before player's action.


NDeck- List of cards in the deck before player's action.


Dealer- Dealer's cards: a list of card predicates.


Table- List of cards held by all players excluding dealer* before
current round (list of lists).












 390playMix( PlayerA, ChDeck, PlayerB, NDeck, Dealer, Table ) :-
 391	playAIDet( PlayerAa, ChDeckA, PlayerB, NDeck, Dealer ),
 392	playAIDeck( PlayerAb, ChDeckB, PlayerB, NDeck, Table),
 393	random(0, 2, Rand),
 394	( (PlayerAa=PlayerAb, ChDeckA=ChDeckB) -> (PlayerA=PlayerAa, ChDeck=ChDeckA)
 395	; Rand=0                               -> (PlayerA=PlayerAa, ChDeck=ChDeckA)
 396	; Rand=1                               -> (PlayerA=PlayerAb, ChDeck=ChDeckB)
 397	).
